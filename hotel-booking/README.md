# Hotel Booking

Aplikasi Go web hotel booking modern (inspirasi Traveloka + Booking.com) dengan:
- 10 hotel seeded.
- Setiap hotel: 10 kamar Biasa, 10 Deluxe, 10 VIP (total 300 kamar).
- Login multi-role (admin/pengguna/staff).
- Payment gateway mock (VA, QRIS, CC, e-wallet).
- Tampilan responsif desktop/mobile/tablet.

## Run
```bash
cd hotel-booking
go run ./cmd/server
```
Lalu buka `http://localhost:8080`.
